## tar

```
*압축하기
tar [옵션] [압축 파일 이름] [압축하고자 하는 파일 or 디렉토리]
tar -cvf test.tar test

*압축풀기
tar [옵션] [압축 파일 이름]
tar -xvf test.tar
```

## tar.gz

```
*압축하기
tar -zcvf test.tar.gz test

*압축풀기
 tar -zxvf test.tar.gz
 ```
